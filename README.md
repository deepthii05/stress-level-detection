# Stress Level Detection

A Streamlit-based Fuzzy Logic Decision Support System that predicts stress level using lifestyle factors such as sleep, workload, screen time, physical activity, and social interaction.

## Features
- Stress score prediction
- Low / Moderate / High classification
- Fuzzy recommendations
- Interactive web interface
- Graphical result visualization

## Technologies Used
- Python
- Streamlit
- scikit-fuzzy
- NumPy
- Matplotlib

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
